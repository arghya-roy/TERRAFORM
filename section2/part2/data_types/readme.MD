### There are mainly four types of data types-
1) string  like "hello"
2) list    like ["mumbai","kolkata","delhi"]
3) map     like {
                  us-east-1 = "t2.micro"
                  us-west-2 = "t2.nano"
                  ap-south-1 = "t2.small"
                     }
4) number  like 200
